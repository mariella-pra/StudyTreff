# StudyTreff 🎓
A web-based community platform designed for students to discover study-friendly locations (cafes, study-spots, bars) and organize academic or social events.

## 🚀 Key Features

* **Dual Content System:** Separate management for permanent "Locations" and time-sensitive "Activities/Events".
* **Interactive Reviews:** Users can rate locations (1-5 stars), leave comments, and upload photos.
* **Event Participation:** Event / Activity joining/leaving system for activities with automated participant counting and capacity limits.
* **Admin Dashboard:** A dedicated management suite to monitor user activity, delete content, and view administrative logs.
* **Profile Customization:** Users can manage their study fields, interests, and profile pictures.
* **Advanced Filtering:** Browse spots by category (Cafe, Study Spot, etc.) or minimum average rating.

## 🛠 Tech Stack

* **Backend:** PHP (Session-based authentication)
* **Database:** MySQL / MariaDB
* **Frontend:** HTML5, CSS3, JavaScript
* **UI Framework:** Bootstrap 5.3
